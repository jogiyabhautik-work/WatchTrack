KEY FEATURES OF THIS APP
=========================
✅ Movie/Anime tracking with watchlists
✅ Theme/Soundtrack discovery (iTunes, MusicBrainz, AnimeThemes)
✅ YouTube audio playback with lyrics support
✅ Watchlist import from files/CSV
✅ User authentication & sync
✅ Theme support (light/dark mode)
✅ Actor/Genre browsing
✅ Manga page flip animation
✅ Binge planning

TECHNOLOGIES USED
==================
- Flutter (Dart)
- Riverpod (State Management)
- HTTP (API Calls)
- SQLite/Appwrite (Backend)
- YouTube Integration
- File Picker
- Web Auth
- Audio Player
- Shared Preferences
